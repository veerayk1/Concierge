# Search

The Search section provides a global search functionality across all platform modules.

**URL**: `/search`

## Search Input

- **Search Bar**: "Type and press enter to search" — full-width text input with search icon
- Results displayed in the right panel under "Search" tab

## Filters (Left Panel)

### Select Building
- Checkbox-based building selection
- Default: "Bond" (checked)

### Search Type
Two search modes available:
1. **Full Search (Default)**: Searches across all content types
2. **Unit/People Search**: Focused search on units and residents

- Dropdown selector to switch between modes

### Search Options (Checkboxes)
All enabled by default in Full Search mode:
- General Logs
- Incident Logs
- Fire Logs
- Noise Logs
- Advertisements
- Maintenance Requests
- Library

## Features
- Cross-module search capability
- Building-specific or multi-building search
- Selectable content type filters
- Two search modes (full vs. focused)
- All search options enabled by default for comprehensive results
