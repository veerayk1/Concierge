# Advertisement

The Advertisement section manages advertisements that can be displayed to building residents.

**URL**: `/advertisements`

## Page Header

- **Title**: "Advertisement" (with megaphone icon)
- **+ Create Advertisements**: Button to create new advertisements (blue/filled)
- **Approve Advertisements**: Button to review and approve pending advertisements (outlined)

## Filters

- **Status Filter**: Dropdown to filter advertisements (default: "All")

## Content Area

- Displays advertisement cards when available
- Shows "No advertisements to show" when empty

## Features
- Advertisement creation workflow
- Approval process for advertisements (suggests moderation/review system)
- Status-based filtering
- Supports advertisement lifecycle management (create, approve, display)
