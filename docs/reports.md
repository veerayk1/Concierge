# Reports

The Reports section provides comprehensive data export and visualization capabilities across all platform modules.

**URL**: `/reports`

## Overview Chart

- **User Breakdown Donut Chart**: Shows distribution of users by type
  - Owners: 235
  - Tenants: 1,298
  - Offsite Owners: 454
  - Board Members: small segment
  - Color-coded legend

## Export Options

Every report supports two export formats:
- **Generate Excel**: Export data as Excel spreadsheet
- **Open PDF Viewer**: View/export as PDF document

(Some reports like Resident List only support Excel export)

## Report Categories

| Report | Description |
|--------|-------------|

### Announcement Reports
| Report | Description |
|--------|-------------|

| Report | Description |
|--------|-------------|

|--------|-------------|
| Shift Report | Security shift reports |

## Features
- Dual export format (Excel + PDF) for most reports
- Category-based organization
- Icon-based visual identification for each report type
- Comprehensive coverage of all platform modules
