# Store (Online Store)

The Store section provides an online marketplace for building-related purchases and services.

**URL**: `/online-store`

## Page Header

- **Title**: "Store" (with grid icon)
- **Select Building**: Dropdown to filter by building (e.g., "Bond")

## Navigation Tabs

| Tab | Description |
|-----|-------------|
| Place Order | Browse and place new orders |
| Existing Orders | View and manage current orders |
| Your Orders | View personal order history |

## Features

- **Cart**: Shopping cart icon with item count badge
- **Photo Size Note**: Info banner stating photos must be under 100kb
- **Payment Integration**: Requires online payment methods to be configured in building settings
- Currently shows: "No appropriate payment methods have been set up in building settings"

## Requirements
- Online payment methods must be configured in building settings before the store can be used
- Product photos must be less than 100kb in size
