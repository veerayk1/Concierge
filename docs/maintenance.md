# Maintenance (Service Requests)

The Maintenance section manages service requests submitted by residents for repairs, maintenance, and building services.

**URL**: `/service-requests`

## Page Header

- **Title**: "Service Requests" (with gear icon)
- **Create Service Request**: Button to submit a new service request

## Search & Filters

- **Search by keyword**: Text search field
- **Status Filter**: Dropdown (default: "All") with clear button
- **Search**: Execute search (red button)
- **Reset Search**: Clear all filters
- **Print Requests**: Print service requests list

## Service Request Cards

Each request is displayed as a card containing:

| Field | Description |
|-------|-------------|
| Title | Request description (e.g., "Microwave Broken") |
| Status | Current status (e.g., "Open") |
| Sent By | Username of the person who submitted |
| Request Type | Category of request |
| Priority | Priority level |
| Posted On | Date and time of submission |
| Assigned To | Staff member assigned to handle |

## Request Types
- **In Suite Repairs**: Repairs inside individual units
- **Service Request**: General building service requests

## Priority Levels
- **High**: Urgent issues
- **Normal**: Standard priority
- **Low**: Non-urgent issues

## Features
- Keyword search across requests
- Status-based filtering
- Priority-based categorization
- Request type classification
- Assignment tracking (who it's assigned to)
- Print capability
- Card-based grid layout (3 per row)
- Date/time tracking for submissions
