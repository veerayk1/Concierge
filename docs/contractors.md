# Contractors

The Contractors section provides a directory of approved contractors and service providers for the building.

**Navigation**: Sidebar link (opens as inline panel, no dedicated URL)

---

## Panel View

- **Title**: "List of contractors for [Building Name]" (e.g., "List of contractors for Bond")
- **Back Button**: Arrow (‹) and "Back" text to return to previous page

---

## Contractor Cards

Each contractor is displayed as a card containing:

| Field | Description |
|-------|-------------|
| Company Name | Name of the contractor/company (e.g., "CITYWIDE LOCKSMITH") |
| Phone Number | Contact phone number (e.g., "905-264-4401") |
| Service Type | Blue hyperlink describing the service category (e.g., "Lock Smith", "Plumber") |

### Service Categories

The contractor list includes various service types:
- **Lock Smith** — Locksmith services
- **Plumber** — Plumbing services
- **Appliance Repair** — Appliance repair and maintenance
- **Flood cleanup services 24/7 and carpet cleaning** — Emergency flood/water damage and carpet cleaning
- **In-suite heating and cooling problems** — HVAC services (resident/owner must make call)
- **For hydro bill and Hydrometers** — Energy management and hydro services

---

## Features Summary

- Building-specific contractor directory
- Contact information for each contractor (company name + phone)
- Service type categorization with descriptions
- Opens as an overlay panel from the sidebar
- Quick access from any page via sidebar navigation
- Read-only directory (no create/edit functionality visible from this view)
