# Survey

The Survey section allows building management to create and distribute surveys to residents.

**URL**: `/surveys`

## Page Header

- **Title**: "Survey" (with clipboard icon)
- **+ Create New Survey**: Button to create a new survey

## Filters

- **Select Building**: Dropdown to filter surveys by building

## Survey Cards

Each survey displays:
- **Survey Icon**: Document icon
- **Title**: Survey name (e.g., "Bond Holiday Party Survey")
- **Status Banner**: Shows if expired with message (orange/yellow banner)
  - "EXPIRED! The survey has expired and is not visible to residents anymore."
- **Visibility Settings**:
  - Visible to Owners: Yes/No
  - Visible to Tenants: Yes/No
- **Dates**:
  - Expiry Date
  - Created On
  - Updated On
- **View Button**: View survey details and responses

## Features
- Survey creation and management
- Building-specific survey distribution
- Audience targeting (Owners, Tenants, or both)
- Expiry date management
- Status tracking (active/expired)
- Survey response viewing
