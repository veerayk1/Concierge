# Library

The Library section serves as a document management system for building-related files accessible to residents and staff.

**URL**: `/library`

---

## Page Header

- **Title**: "Library" (with document icon)
- **Create New Folder** button — creates a new document folder

## Search & Filters

| Filter | Type | Description |
|--------|------|-------------|
| Search File | Text input | Search for files by name |
| Search Folder | Text input | Search for folders by name |
| Select Building | Dropdown | Filter by building |

## Statistics Cards

| Card | Description |
|------|-------------|
| Files Uploaded | Total number of uploaded files (e.g., 23) — "All Time" |
| Total Space Used | Total storage consumed (e.g., 10.02 MB) — "All Time" |

---

## Folder Structure

### Folder Actions
Each folder has four action buttons:

| Button | Action |
|--------|--------|
| **Update** | Edit folder name/description |
| **Delete** | Delete the folder |
| **Permissions** | Manage role-based access permissions |
| **+ Add File** | Upload a new file to the folder |

### File Actions
Each file within a folder displays:
- **Filename** (clickable link)
- **Download** button
- **Delete** icon
- **Added On** date

### Folders (8 total)

1. **Expert Tips** — Tips and information from experts
2. **Documents** — General building documents
3. **Forms** — Official building forms (Party Room Rental Agreement, Guest Suite Reservation, Elevator Agreement, Action Alert Form)
4. **Notices** — Bond Notices (Pet Etiquette Reminders, Maintenance Schedules, Construction Updates, Accessibility Notices, Public Hearing Notices, Rule Changes, Disclosure Forms)
5. **Minutes** — Meeting minutes and records
6. **Newsletters** — Building newsletters
7. **Rules** — Building rules and regulations
8. **Suite plans** — Suite/unit floor plans

---

## Permissions Modal

Opened by clicking **"Permissions"** on any folder.

Manages role-based access control with two permission levels per role:

| Permission | Description |
|------------|-------------|
| **Read** | View/download files in the folder |
| **Read-Write** | View, download, upload, and delete files |

### Role Groups (14 total)

| Role Group | Description |
|------------|-------------|
| Admin | System administrators |
| Property Manager | Property management staff |
| Family member - Spouse | Spouse family members |
| Family Member - Child | Child family members |
| Other Occupant | Other unit occupants |
| Tenant | Tenants |
| Owner | Unit owners |
| Offsite Owner | Owners not residing on-site |
| Other Group | Other role groups |
| Superintendent | Building superintendent |
| Supervisor | Supervisors |
| Security Head Office | Security head office staff |
| Board Member | Board members |
| Security Guard | Security guards |

Each role group has individual **Read** and **Read-Write** checkboxes.

---

## Features Summary

- Document storage and management organized by folders
- File upload via drag-and-drop or click (+ Add File per folder)
- File download and deletion
- Folder creation, update, and deletion
- Full-text search for files and folders
- Building-specific filtering
- Role-based access control with 14 role groups
- Two permission tiers: Read and Read-Write
- Storage usage tracking (file count and space used)
- Supports multiple file types (PDF, DOCX, etc.)
- Timestamp tracking for all uploads
