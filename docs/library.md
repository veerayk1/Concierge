# Library

The Library section serves as a document management system for building-related files accessible to residents and staff.

**URL**: `/library`

## Page Header

- **Title**: "Library" (with document icon)
- **Create New Folder**: Button to create new document folders

## Search & Filters

- **Search File**: Text search for files
- **Search Folder**: Text search for folders
- **Select Building**: Dropdown to filter by building

## Statistics Cards

| Card | Value | Description |
|------|-------|-------------|
| Files Uploaded | 23 | Total number of uploaded files |
| Total Space Used | 10.02 MB | Total storage consumed |

Both cards show "All Time" metrics.

## Folder Structure

Each folder contains:
- **Folder Title** (numbered)
- **Description** of folder contents
- **Action Buttons**: Update, Delete, Permissions, + Add File
- **File Listings** with: filename (clickable), Download button, Delete icon, "Added On" date

### Folders

#### 1. Expert Tips
- Description: "Tips and information from experts"
- Contains PDF files with building tips and guides

#### 2. Documents
- Description: "Documents"
- Contains general building documents (.docx, .pdf)

#### 3. Forms
- Description: "Forms"
- Contains official building forms:
  - Party Room Rental Agreement
  - Guest Suite Reservation
  - Elevator Agreement
  - Action Alert Form

#### 4. Notices
- Description: "Bond Notices"
- Contains building notices and memos:
  - Pet Etiquette Reminders
  - Maintenance Schedules
  - Construction Updates
  - Accessibility Notices
  - Public Hearing Notices
  - Rule Changes
  - Disclosure Forms

## File Features
- File download capability
- File deletion
- File upload (+ Add File per folder)
- Folder-level permissions management
- Folder update and deletion
- Supports multiple file types (PDF, DOCX)
- Timestamp tracking for all uploads
