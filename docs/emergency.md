# Emergency

The Emergency section displays building-specific emergency procedures in a modal/popup overlay.

**Navigation**: Sidebar link (opens as modal, no URL change)

## Modal View

- **Title**: "Emergency Procedure for [Building Name]" (e.g., "Emergency Procedure for Bond")
- **Back Button**: Close the modal and return to previous page
- **Content Area**: Displays emergency procedure text/instructions
  - Currently shows "null" (no emergency procedures have been configured for Bond)

## Features
- Building-specific emergency procedures
- Opens as a modal overlay on top of current page
- Quick access from any page via sidebar
- Supports rich text content for emergency instructions
- No dedicated URL — triggered via JavaScript from sidebar
