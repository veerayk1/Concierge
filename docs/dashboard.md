# Dashboard

The main landing page of the Aquarius Condo Management Portal providing an at-a-glance overview of the property.

## Header Section

- **Property Info**: TSCC 2584 - Toronto
- **Date/Time Display**: Shows current date and time
- **Weather Widget**: Displays current temperature, wind speed, and weather condition (mood)

## Quick Action Buttons

- **Reset View**: Resets the dashboard view with an external link icon
- **Contacts**: Quick access to contacts
- **+ Shift Logs**: Button to add new shift logs (blue button, top right)
