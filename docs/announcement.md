# Announcements

The Announcements section manages building-wide communications to residents.

**URL**: `/announcements`

## Page Header

- **Title**: "Announcements"
- **Manage Custom Groups**: Button to manage recipient groups
- **Create New Announcements**: Button to create a new announcement

## Search & Filters

- **Search**: Text search field
- **Select Building**: Dropdown to filter by building
- **Search / Clear Search**: Action buttons
- **Print**: Print announcements

## Announcement Cards

Each announcement is displayed as a card containing:
- **Title**: Announcement heading (e.g., "Women's Steam room - Out of Service")
- **Building**: Which building it applies to (e.g., "Bond")
- **Expiry**: Expiration setting ("Never Expiring" or specific date like "Expiring On: 05-04-2026 00:00")
- **Posted By**: Author name
- **Posted On**: Date posted (e.g., "11-03-2026")

### Card Actions
- **Email Failures**: View failed email deliveries for this announcement
- **Mailing List**: View the distribution list
- **View**: View full announcement details

## Announcement Types

### Active Announcements
- Standard announcements visible to all residents
- Can be set to "Never Expiring" or with a specific expiry date

### Scheduled Announcements
- Highlighted with purple "Scheduled For Later" badge
- Shows scheduled send date/time (e.g., "Scheduled to be sent on: 09-03-2026 11:00")
- Includes verification note about checking attachments before sending

## Features
- Multi-building support with building filter
- Scheduled announcement delivery
- Expiry management (never expiring or date-based)
- Email delivery tracking (failures, mailing lists)
- Custom group management for targeted communications
- Print functionality
- Grid layout (3 cards per row)
